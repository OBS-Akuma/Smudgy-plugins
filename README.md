## What is this repo for?

This repository serves for the **Install tab** inside the Smudgy Client menu. 

Users can:
- View code updates
- Download new code
- See username, timestamp, type, and name for each submission

---

## Features
- Real-time code browsing
- User attribution for each script
- Easy installation directly from the client

---

## Getting Started
1. Open Smudgy Client
2. Navigate to the **Install** tab
3. Browse available scripts
4. **Read the code before installing**
5. Click to install your desired plugin

---

## Security Notes
- Community scripts are **not officially vetted**
- Token grabbers and malicious code may be present
- You are responsible for reviewing any code you run
- If unsure about a script, do not install it
